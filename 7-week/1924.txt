2007년 x월 y일의 요일을 구하는 문제
1월 1일은 월요일이다.

2007년의
1, 3, 5, 7, 8, 10, 12월은 31일까지
4, 6, 9, 11월은 30일까지
2월은 28일까지 있다.

각 월의 1일의 요일을 구함
해당 요일은 각 달에 대한 case로 숫자로 저장 ex) 월요일 -> 1
주어진 일수(y)를 우선 7로 나누어 나머지 값을 구함
나머지 값을 요일(숫자로 저장된)과 합함 
해당 값을 case로 출력 ex) 1 -> 월요일, 2 -> 화요일 ...



월(x)과 일수(y)를 입력
해당하는 월(x)의 case(switch사용)로 이동
case로 넘어갔을 때 해당 월(x)의 일수(y)의 범위가 올바른지 확인
value에 해당하는 월(x)의 요일값(월요일->1,화요일->2...일요일->7)으로 초기화
value에 일수(y) % 7 -> 나머지 값을 합함
숫자에 해당하는 요일을 출력
